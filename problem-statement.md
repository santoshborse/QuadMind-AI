

In enterprise setting a employee may receive multiple messages in a day may be in from Slack, email, text or other medium of communication.

Manytime these messages also contains a action item or todo, it may be a very small like responding with yes or no

or it could be a big task like writing a report or sending a email to a client.


Requirements:
- Create a ssystem which monitors Slack messages 
- Create a dashboard of todo list
- the dashboard is auto updated when a new message is received
- the dashboard should be able to mark task completed as the responded to the message
- Dashboard has ability to arrange by priority or by cronological order
